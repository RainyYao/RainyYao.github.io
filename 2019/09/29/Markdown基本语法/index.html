Markdown基本语法

发表于 2019-09-29

Markdown 是一种轻量级的标记语言，它允许人们使用易读易写的纯文本格式编写文档，当前很多网站都广泛使用Markdown来撰写帮助文档或是用于论坛上发表消息。Markdown 语言在2004年由约翰·格鲁伯（John Gruber）创建。Markdown 编写的文档可以导出HTML、Word、图像、PDF、Epub等多种格式的文档。Markdown 编写的文档后缀为.md，.markdown。

编辑器

本人使用的是Typora编辑器。它是一款十分简洁方便的Markdown编辑器，完全免费且支持中文。可以在windows、mac、linux平台使用。界面简洁无多余功能，提供了很多快捷键和格式编辑选项，界面所见即所得。

标题

格式：#+空格+内容

1个#是1级标题，2个#是2级标题，以此类推。支持六级标题。
